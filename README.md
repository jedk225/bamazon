# bamazon

```
Bamazon is an Amazon-like storefront utilizing Node.js & MySQL
```

# Dependencies
```
1. mysql
2. cli-table
3. inquirer

Please install each of the packages listed above by typing 'npm install [Package Name]' in the command line.
```
